# Galt 产品文档

> 基于仓库现状梳理 ｜ 平台 macOS 14+ ｜ 形态 菜单栏常驻应用（`LSUIElement`）
>
> 本文以**当前代码实现**为准，描述 Galt 现在是什么、能做什么、核心亮点与架构。
> 历史里程碑式需求见 [PRD.md](../PRD.md) 与 [APP_SPEC.md](../APP_SPEC.md)（注意：这两份偏早期，部分内容已落后于现状——以本文为准）。

---

## 1. 一句话定义

Galt 是一款常驻 macOS 菜单栏的 AI 语音听写工具：**按住快捷键说话，松开后自动「转写 → 润色 → 注入光标」**，全程无需打开窗口。它不是逐字转写，而是把口语整理成可直接发送的成稿。

## 2. 定位与目标用户

- **品类**：菜单栏实用工具，不在 Dock 常驻、无主窗口打扰，日常只在屏幕底部用一个悬浮胶囊与你交互。
- **核心主张**：零摩擦、成稿级、可离线、数据自主。

| 目标用户 | 核心诉求 |
|---|---|
| 知识工作者 / 写作者 | 口述替代打字，快速产出邮件、文档、聊天 |
| 开发者 | 中英混排、保留技术术语、注入 IDE / 终端 |
| 多语言用户 | 任意语言口述，输出指定目标语言成稿 |
| 隐私敏感用户 | 全程本地离线识别，数据不出设备 |

---

## 3. 核心亮点

1. **零摩擦的全局热键** —— 一个键完成「想说→说出→成稿」。支持**按住说话**（hold-to-talk，松开即出字）与**轻点锁定**（tap-to-lock，免按住长录音，再点结束）。三种模式各自可绑定组合键。
2. **成稿级输出，而非逐字转写** —— LLM 后处理去口水词（嗯/呃/那个）、识别口头自我纠正、按上下文纠同音错字、自动标点与结构化，**保留原意与语气、拒绝扩写与编造**。
3. **上下文感知** —— 按前台 App 类别自动调语气：邮件正式成文、聊天简短随意、代码保留英文标识符、笔记倾向列表分段。
4. **纠错自学习** —— 出字后观察你的就地修改，用编辑距离检测词级纠正（如 galt→Galt），把高频纠正沉淀进个人词典，越用越准。
5. **三种模式 + 语音编辑** —— 听写 / 翻译 / 随便问（结果进浮动卡片，不污染光标）；选中文字后口述指令即可改写（「改短一点」「换正式语气」「翻成英文」）。
6. **引擎双轨、可完全离线** —— 云端 5 家厂商 + 本地 3 套引擎，`auto` 模式云端优先、失败自动回退本地；隐私敏感场景可切「仅本地」。
7. **流式与压缩** —— DashScope / 火山引擎支持流式 ASR；上传走 Opus（~10×）/ AAC 压缩，降低体积与时延。
8. **标志性 HUD 反馈** —— 屏幕底部悬浮胶囊，实时声波 + 红点律动、唤醒/录音/处理进度/成功/失败/取消等完整状态、倒计时、ESC 随时取消。
9. **隐私优先** —— 听写历史与词典仅存本地，API Key 进 Keychain，云端调用前明确告知数据离开设备。
10. **原生桌面体验** —— SwiftUI + AppKit，菜单栏常驻，遵循 macOS HIG，自适应深浅色，VoiceOver 可访问，尊重「减弱动态效果」。

## 4. 核心交互

### 4.1 三种模式

| 模式 | 行为 | 结果去向 |
|---|---|---|
| **听写**（dictation） | 转写 + 可选润色 | 注入当前光标 |
| **翻译**（translate） | 转写 + 翻译到目标语言（简中 / 英 / 日） | 注入当前光标 |
| **随便问**（ask） | 转写为问题 → LLM 作答 | 浮动结果卡片（可复制 / 插入 / 关闭） |

### 4.2 触发方式

- **按住说话**：按住绑定键说话，松开→转写→注入。
- **轻点锁定**：快速点按（< 阈值）进入锁定录音，适合长段落，再次点按或点 HUD「✓」结束；单次会话上限 6 分钟，临近时 HUD 显示倒计时。
- **快捷键绑定**：可选 fn / 左右 ⌘ / 左右 ⌥ / 左右 ⌃ / 右 ⇧ 的单键或组合键，三模式各绑一套。无辅助功能授权时回退 Carbon 组合键（⌃⌥⌘Space 等）。

### 4.3 语音编辑

听写模式下若有选中文本，则把口述当作**改写指令**作用于选区（其余模式忽略选区）。

### 4.4 文本注入与撤销

剪贴板暂存 → ⌘V → 还原剪贴板，注入任意 App 光标处；有辅助功能权限时走逐字流式键入（边生成边出字）。成功态提供「撤销」，失败保留音频可「重试」。

## 5. 智能层（LLM 后处理）

三种后处理模式（`Polisher`），统一走 OpenAI 兼容 `/chat/completions`，支持流式：

- **润色 polish**：最小化清理（去填充词、自我纠正、同音纠错、标点、结构化），保留语言/语气，拒绝扩写与执行。
- **翻译 translate**：润色后翻译到目标语言，只输出译文。
- **作答 ask**：直接回答口述问题，简洁、同语言、无寒暄。
- **编辑 edit**：按指令改写选中文本，严格遵循指令。

词典与上下文：把「手填词典 + 自动学习词」合并注入 prompt 指导用词；按前台 App bundle id 映射语气规则（邮件/IM/代码/文档）。

## 6. 转写引擎与供应商矩阵

引擎抽象为 `STTProvider` 协议，`engineMode` 决定路由：`auto`（云端优先、失败回退本地）/ `cloud` / `local`。

### 6.1 云端 STT（5 家）

| Provider id | 名称 | 协议 | 音频编码 |
|---|---|---|---|
| `groq` | Groq Whisper | REST（OpenAI 兼容） | AAC/m4a，回退 WAV |
| `siliconflow` | 硅基流动 SenseVoice | REST | AAC/m4a，回退 WAV |
| `openai` | OpenAI | REST | AAC/m4a，回退 WAV |
| `dashscope` | 阿里云百炼 ASR | HTTP 或 **WebSocket 流式** | WAV |
| `volcano` | 火山引擎 | HTTP / **WebSocket 流式** / 文件异步 | **ogg_opus** 压缩，回退 WAV |

DashScope 预设含 Qwen3-ASR、Fun-ASR、Paraformer（含 8k）等实时模型；火山含极速版 / 流式大模型 / 标准版。

### 6.2 本地 / 离线引擎（3 套）

| 引擎 | 说明 | 模型 |
|---|---|---|
| **Apple 设备端听写** | 零下载、需指定语言、首次请求「语音识别」权限 | 系统内置 |
| **whisper.cpp** | Metal GPU 加速、自动检测语言 | 6 档量化模型 tiny→large-v3（30MB→1GB），应用内下载，默认 small-q5_1 |
| **sherpa-onnx** | ONNX Runtime（`GALT_SHERPA` 已启用） | SenseVoice（多语种）/ Paraformer（中文长句） |

### 6.3 LLM 供应商（6 家，润色/翻译/作答/编辑）

`groq` · `dashscope`（通义千问） · `ark`（火山方舟豆包） · `deepseek` · `siliconflow` · `openai`，均为 OpenAI 兼容协议，模型与超时可在「模型库」逐项配置。

### 6.4 音频编码

WAV（16k 单声道）为基线；OpenAI 兼容厂商优先 AAC/m4a；火山可选 Opus（`COpusShim` + libopus，RFC 7845 Ogg 封装，~10× 压缩）。

## 7. HUD 状态（屏幕底部悬浮胶囊）

| 状态 | 呈现 |
|---|---|
| 唤醒中 | 启动麦克风的过渡态（异步启动 + 防抖，启动够快则不闪） |
| 录音中 | 红点律动 + 实时声波 + 取消 ✕ / 完成 ✓；锁定态带倒计时；前几次带教学提示；ESC 取消 |
| 处理中 | 「转写中…/润色中…」+ 预测进度条（0→96%，完成补满） |
| 成功 | 「已插入文本 / 已复制到剪贴板」，短暂停留可撤销 |
| 失败（可重试） | 警示 + 「重试」+「关闭」，保留音频 |
| 取消 / 空结果 / 提示 | 「已取消」/「未捕获到语音」/「已学习『词』」轻量反馈后淡出 |

模式以颜色区分（听写红 / 翻译青 / 问答蓝 / 编辑橙）；底部居中跟随鼠标所在屏；全程尊重减弱动态效果，状态变化向 VoiceOver 播报。

## 8. 应用形态

- **菜单栏**：图标三态（录音红 / 暂停灰 / 空闲）；菜单含使用提示、累计统计（次数/字数/WPM/省时）、打开控制台、暂停听写、AI 润色开关、翻译子菜单、麦克风设备子菜单、设置、打开历史、权限与麦克风自检、重跑引导、退出。
- **控制台窗口**（单一主窗口）：概览（统计卡 + 生产力趋势 + 应用占比 + 最近转录）、历史（筛选/分页/播放/复制/删除，保存周期可设）、词典（自动学习 / 手动添加，分段筛选 + 搜索）。
- **设置**（控制台内二级路由）：键盘快捷、语言、音频、语音引擎（策略与开关）、模型库（凭证与模型/本地模型下载）、通用（外观/登录启动/Dock/纠错自学习/润色/最长录音）。
- **引导**（首次启动）：欢迎 → 权限（麦克风/辅助功能/语音识别）→ 偏好（快捷键/语言/模式）→ 练习（打开示例 App 实测首次听写）。
- **诊断窗口**：权限与麦克风自检——授权状态、实时电平表、跳转系统设置、`tccutil` 重置授权。

## 9. 隐私与数据

| 数据 | 位置 |
|---|---|
| 听写历史 | `~/Library/Application Support/Galt/`（JSONL，仅本地；音频按设置可选留存并按期清理） |
| 个人词典 | 本地（注入转写 / 润色 prompt） |
| 应用设置 | UserDefaults |
| API Key / 密钥 | Keychain 加密（旧明文自动迁移），禁止日志输出 |

云端 STT / LLM 需用户自带凭证；`auto` 模式可从云端回退本地；UI 文案明示数据何时离开设备。

## 10. 技术架构（概览）

- **语言/框架**：Swift（SwiftUI + AppKit），SwiftPM；macOS 14+；Bundle `com.sorcerer.galt`。
- **二进制依赖**：whisper / sherpa-onnx / onnxruntime / opus（xcframework，按需 `scripts/fetch-vendor.sh` 获取）+ `COpusShim`（libopus C 桥）。
- **关键模块**：`DictationController`（编排）、`HotkeyManager`（全局热键，CGEvent tap + 看门狗）、`AudioRecorder`（AVAudioEngine 采集 + 感知电平）、`HUDController`（NSPanel 胶囊）、`STTProvider`/`CloudProviders`（引擎路由与厂商）、`Polisher`（LLM）、`EditLearner`（纠错自学习）、`TextInjector`/`SelectionReader`（注入与选区）、`HistoryStore`（历史与统计）、`SettingsStore`/`KeychainStore`（设置与密钥）、`Tuning`（手感常量）。
- **构建**：`make build`（swift build）、`make vendor`（取二进制框架）、`make app` / `make dmg`（打包与分发）。CI（GitHub Actions）在 push/PR 跑 `swift build`。
