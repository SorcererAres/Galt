# Galt — Typeless 1:1 复刻计划（定稿）

> 目标：在交互与功能层面完整复刻 Typeless（AI 语音听写工具），代码与视觉为原创实现，不使用 Typeless 的商标、图标或素材。项目代号 **Galt**。

## 已确认的技术决策

| 决策点 | 结论 |
|---|---|
| 桌面端技术栈 | **Swift 原生**（SwiftUI + AppKit，菜单栏常驻 App） |
| 语音转写引擎 | **云端 + 本地双引擎**：默认 Groq Whisper API，本地引擎第一版采用 **Apple 设备端语音识别**（系统原生、零下载、完全离线），whisper.cpp 留作 M4 可选增强 |
| 第一期范围 | **仅 macOS**，官网与 iOS 延后 |

## 产品功能清单（对标 Typeless）

**核心闭环**
1. 全局按住 `fn` 即说（hold-to-talk），松开即出字，全程无需打开窗口
2. 录音时屏幕底部悬浮波形胶囊（HUD），实时反馈状态
3. 两段式管线：STT 转写 → LLM 润色（去填充词、识别口头自我纠正、自动标点、口述列表结构化）
4. 文本自动注入当前光标所在输入框（任意 App 通用）

**智能层**
5. 上下文感知：检测前台 App，自动调整语气与格式
6. 语音编辑：选中文字 + 说指令直接改写
7. 多语言自动检测 + 翻译模式
8. 个人词典 + 常用词汇自动学习

**外围**
9. 听写历史（仅本地）+ 统计（WPM、节省时间）
10. 设置界面（快捷键、语言、语气预设、词典）
11. Onboarding 权限引导（麦克风 + 辅助功能）
12. 锁定模式（点按开始/结束）、单次会话 6 分钟上限

## 架构

```
Galt.app（菜单栏 App，LSUIElement）
├─ HotkeyManager        全局 fn 键监听（NSEvent 监视器）
├─ AudioRecorder        AVAudioEngine 采集 → 重采样 16kHz 单声道 WAV
├─ HUDController        无边框 NSPanel + SwiftUI 波形胶囊
├─ STTProvider 协议      ├─ GroqWhisperProvider（云端，默认）
│                       └─ LocalWhisperProvider（whisper.cpp，M3 接入）
├─ Polisher             LLM 后处理（注入前台 App 上下文 + 个人词典）
├─ TextInjector         剪贴板暂存 → ⌘V 粘贴 → 恢复剪贴板
├─ SettingsStore        设置持久化（API Key 后续迁移 Keychain）
└─ HistoryStore         本地历史与统计（M3）
```

## 运行要求

- macOS 14+，需授权「麦克风」与「辅助功能」
- 云端引擎需 Groq API Key（菜单栏 → 设置 API Key，或环境变量 `GROQ_API_KEY`）
- 建议将 系统设置 → 键盘 →「按下 🌐 键时」设为「无操作」，避免与听写热键冲突
