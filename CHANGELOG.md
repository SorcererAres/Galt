# 更新日志

本文件记录 Galt 的重要变更。格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

## Unreleased

### 新增

- **纠错自学习**：出字后观察用户的就地修改，把干净的单词级纠正（如 `galt → Galt`、`github → GitHub`）自动并入个人词典，同时增强转写提示与 LLM 润色；纯中文同音改动、整段重写不纳入。学到的词在控制台「词典 → 自动」可见、可单独删除。
- **历史保存音频**：可选随历史在本机保存录音（默认关闭），便于回放与重新转写；新增「检测语种 / 状态」元数据，删除与超期清理会连同音频一并清除。
- **权限与麦克风自检面板**：菜单栏新增入口，含**实时麦克风电平条**（一眼确认是否真的在拾音）、麦克风与辅助功能权限状态及快捷操作（请求授权 / 打开系统设置 / 重置辅助功能授权）。
- **全局暂停开关**：菜单栏「暂停听写」，暂停时菜单栏图标转灰、热键被忽略，便于全屏 / 游戏 / 会议时避免误触。
- **「随便问」结果卡片**：ask 模式的答案改用浮动卡片流式呈现，支持复制 / 插入到光标 / 关闭，不再直接键入光标。
- **引导「试一试」步骤**：新用户引导结尾新增实地练习步骤，可一键打开常用 App（备忘录 / Safari / 邮件 / 微信）并实时检测首次听写是否成功。

### 改进

- **上传音频压缩**：
  - 火山「录音文件」识别（极速版 / 标准版）上传前用 **Opus（ogg_opus）压缩，体积约降 10×**，编码失败自动回退 WAV；可在火山设置中开关（vendor libopus + 自实现 Ogg 封装）。
  - OpenAI 兼容厂商（Groq / OpenAI / 硅基流动）上传改用原生 **AAC（m4a）压缩**。
- **系统中断自动停录**：录音途中遇到锁屏 / 休眠 / 屏保时丢弃本次采集，避免录音悬挂或误转写环境音。
- **个人词典生效面更广**：手填词典与自动学习词合并后统一喂给转写提示与 LLM 润色。

### 修复

- **火山错误提示**：验证 / 转写失败时显示火山真实状态码与消息，而非笼统的「HTTP 200」。
- **火山「无有效语音」**：状态码 `20000003`（静音 / 无有效语音）按空结果优雅收尾，不再误报为错误；真静音录音也会得到「未捕获到语音」提示而非报错。

### 维护

- 新增仓库基础文件：忽略产物配置、贡献指南与更新日志。
